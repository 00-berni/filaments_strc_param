# Filament Recostruction

## Requirements

  * [DisPerSE](https://www.iap.fr/useriap/sousbie/web/html/indexd41d.html) is required. Install it from [FranUcles](https://github.com/FranUcles/DisPerSE/tree/master)
  * Anaconda
  * Python3.9

## Instalation

```bash
$ conda create --name [env-name] --file requirements.txt
```
